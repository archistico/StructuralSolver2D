# StructuralSolver2D documentation package

This package contains the initial planning documents for an independent .NET/C# planar structural analysis engine.

The project is intentionally separated from OpenCad2D until the calculation engine is stable and validated.

Documents:

- `docs/structural/vision.md`
- `docs/structural/scope.md`
- `docs/structural/architecture.md`
- `docs/structural/model.md`
- `docs/structural/units.md`
- `docs/structural/analysis.md`
- `docs/structural/validation.md`
- `docs/structural/roadmap.md`
