# Interactive HTML viewer prototype

Milestone 47 introduces a first lightweight interactive viewer prototype.

The viewer is intentionally implemented as a standalone HTML export rather than a desktop UI project. This keeps the solver independent from WPF, Avalonia or browser-specific application frameworks while still providing an immediately usable inspection surface.

## Export command

```powershell
dotnet run --project src\StructuralSolver2D.Cli -- export-viewer examples\beams\simple-supported-beam.json reports\viewer\simple-supported-beam.html
```

Optional final argument:

```text
[loadCaseId|combinationId]
```

Example with a combination:

```powershell
dotnet run --project src\StructuralSolver2D.Cli -- export-viewer examples\combinations\load-combination.json reports\viewer\combination.html ULS1
```

## Features

The first prototype supports:

- mouse wheel zoom;
- pointer drag pan;
- reset view;
- layer toggles for undeformed model, deformed shape, diagrams, supports, reactions and dimensions;
- summary values for nodes, members, supports, reactions, diagrams and deformation scale.

The viewer reuses the same SVG result geometry produced by the reporting visualization layer.

## Architecture

The viewer is generated by:

- `InteractiveHtmlStructuralViewerExporter`;
- `InteractiveViewerExportOptions`.

It consumes only a `StructuralVisualizationModel`.

The viewer does not contain solver logic and does not modify the structural model.

## Intentional limitations

This first prototype does not yet provide:

- editable model geometry;
- dynamic recomputation;
- diagram scale sliders;
- animation playback controls;
- point inspection tooltips;
- OpenCad2D integration.

Those features remain reserved for later milestones.
